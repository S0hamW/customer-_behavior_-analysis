Overview
This project demonstrates a complete end-to-end data analytics workflow, starting from raw data to actionable business insights. It includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, interactive dashboard development, and presentation of findings.

The objective of this project is to transform raw data into meaningful insights that support data-driven decision-making.

Project Steps

1. Data Loading
Imported dataset using Python.
Verified data types and dataset structure.

3. Exploratory Data Analysis (EDA)
Dataset overview
Summary statistics
Missing value analysis
Duplicate detection
Distribution analysis
Correlation analysis
Initial visualizations

4. Data Cleaning
Handled missing values
Removed duplicates
Corrected data types

Prepared data for analysis

5. SQL Analysis

The cleaned dataset was imported into PostgreSQL for advanced querying.

Business questions answered include:
